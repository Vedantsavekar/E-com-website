This is a fully functional E-commerce website developed using HTML, CSS, JavaScript, PHP, and MySQL.

🔹 Features:
- User registration & login system (with password hashing)
- Session-based authentication
- Product upload and display from database
- Add to cart functionality using PHP sessions
- Cart management (increase, decrease, delete items)
- Product view page with details
- Secure database connection using MySQL

🔹 Technologies Used:
- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL

🔹 Purpose:
This project is built to demonstrate full-stack web development skills including frontend design, backend logic, and database integration.
